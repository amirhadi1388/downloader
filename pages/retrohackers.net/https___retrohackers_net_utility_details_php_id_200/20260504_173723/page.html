The best Zelda III editorSunGodPortal2016-01-29Version 0.964

Hyrule Magic is a great program. It can edit just about anything in the game (except for some of the ending text which has to be edited via hex editor).

It has an overworld editor, dungeon editor, title screen and menu editor, music editor, palette editor and even a "monologue" (!) editor for editing the dialogue.

It can't edit graphics but you can copy graphics from the game into the clipboard, edit them in another program and them import them back into the game with Hyrule Magic. This is handy since the majority of the graphics are compressed and would otherwise need to be extracted/decompressed, edited and then reinserted/recompressed via the archaic, command-driven program known as Zcompress.

Hyrule Magic is very easy to use and most of all, it's fun!

Thank you, Sephiroth3!!!

Nice ProgramDogeritos2015-02-03Version 0.962

This program is pretty good. I'd consider it user-friendly. All a beginner needs to do is either look up a tutorial on the internet, or the Help section of the program itself. I'd say it's definitely better than Black Magic... You can edit dungeons, the overworld, palettes, and on version .963, they make editing the other graphics easier (although YY-CHR is the way to go for your graphics editing). Sometimes, it'll do some funky glitches, but they hardly ever happen. All on all, it's a good program, for beginners, and intermediate Zelda 3 hackers.
